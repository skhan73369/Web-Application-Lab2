What to expect

25 technical sessions

The main track is 25 exciting technical sessions, covering everything from performance, accessibility, testing, and security using the latest tools and APIs for building Progressive Web Apps.

Add payments and virtual reality to your sites, and build modern layouts with new CSS features like CSS Grid.

Tracks

This year, you can meet with engineers from Microsoft, Google, Facebook, and Twitter who can help you to solve real problems today, and build invaluable connections for the future. Looking to reduce a troublesome performance issue? Struggling with best practices for accessibility? Eager to get started with VR? Curious about Angular and React? We've got you covered. These tracks connects you one-to-one with engineers throughout the day and get results you can take back to your site or app.

Debug

Struggling with JavaScript bugs? Looking for test guidance? Need help with legacy compatibility? Our engineers are passionate about making the web work for all our mutual customers, and we want to help. Grab a drink at the Debug Bar and we'll take a look!

Performance

Web performance is hard but we can help! Show us your site and we'll apply the same tools we use to build our advanced applications. Our team has years of experiencing helping properties around the world identify and remediate performance pitfalls across browsers, so your customers have a better experience on every device. We'll provide actionable recommendations for your site, and opportunities to follow up with a deeper conversation with our developers.

Security

New security tools are here! Learn the new specification and how to apply them to make your web apps hack proof. Come try out live attack lab and see how your site holds up. Learn more about building sites which incorporate authentication and authorization on both the client and server and see how you can get started developing the most secure apps on the planet - no previous experience required!

And More

That's just the beginning! We're partnering with teams across the industry to deliver opportunities for trying out new experiences and learn about new tools. We'll have representatives from GitHub, Visual Studio, TypeScript, the brand-new .NET Core 2.0 framework, and more
